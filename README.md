## Description
This Python script utilizes `pyttsx3` to convert text into speech, creating a text-to-speech robot speaker. It provides a simple command-line interface for entering text, and the robot speaker articulates it instantly. The script uses `tkinter` to display a hidden message box for a seamless user experience.

## Features
- Interactive: Input any text, and the robot speaker will articulate it.
- Error Handling: Graceful handling of text-to-speech errors.
- User-Friendly: Hidden message box ensures a clean display.

## Requirements
- Python 3.x
- `pyttsx3` library

## Usage
1. Install Python 3.x and `pyttsx3`.
2. Clone the repository or download `robot_speaker.py`.
3. Run the script:
4. Enter the text you want the robot speaker to say and press Enter.
5. To exit, type 'exit' when prompted.
